# Your Hoop Tracker

Your Hoop Tracker is a simple basketball session tracking app. The goal of the app is to help organize local basketball runs by allowing players to view sessions, sign up for a run, and help the organizer track who has joined.

## Project Goal

The main purpose of this project is to make it easier to manage basketball sessions. Instead of keeping track of players through texts, group chats, or random notes, this app gives players and organizers one place to manage signups.

## Planned Features

- User sign up and login
- View available basketball sessions
- Join a session
- Track confirmed players
- Show session details such as time, location, price, and available spots
- Admin view for managing player lists
- Payment status tracking for each player

## MVP Features

The first version of the app will focus on the core features:

- Display tonight’s basketball run
- Allow a user to join the run
- Add the user to a player list
- Show how many spots are filled
- Allow an admin to mark players as paid or unpaid

## Tech Stack

This project is planned to use:

- React
- Vite
- Tailwind CSS
- Supabase for authentication and database features later

The first version may use mock data before connecting to a real database.

## Current Status

This project is currently in the early development stage. The first focus is building the main user flow:

1. Login or sign up
2. View tonight’s run
3. Join the run
4. View the confirmed player list
5. Track payment status

## Future Improvements

Possible future features include:

- Real user accounts
- Online payments
- Multiple basketball sessions
- Player profiles
- Waitlist system
- Session history
- Notifications
- Mobile-friendly design improvements

## Why I’m Building This

I am building this app to help make organizing basketball runs easier for my community. The app is meant to reduce confusion, keep track of who is playing, and make it easier to manage spots and payments.
